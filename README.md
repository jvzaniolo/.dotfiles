- Git config

## Installation
First, [set up ssh](https://docs.github.com/pt/github/authenticating-to-github/connecting-to-github-with-ssh).

Clone this project
```bash
git clone git@github.com:jvzaniolo/.dotfiles.git ~/.dotfiles
```

And run the install command
```bash
~/.dotfiles/install
```
